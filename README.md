## Get Started

Create new project

```shell
$ mkdir gradle-get-started

$ cd gradle-get-started

$ gradle init
```
this will generate some fils under gradle-get-started

```shell
$ tree

.
├── build.gradle
├── gradle
│   └── wrapper
│       ├── gradle-wrapper.jar
│       └── gradle-wrapper.properties
├── gradlew
├── gradlew.bat
└── settings.gradle
```
